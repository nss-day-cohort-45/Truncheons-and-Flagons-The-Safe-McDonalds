Buttons: Use div - class="btn". By default, font-family is same as headings. Style with BEM modifiers as needed.
Select: By default, select element inherits font styling from its parent. Use an option element with value of zero for the default option
Text input: By default, input elements inherit font styling from their parent. Be sure to use a placeholder attribute and/or a label for clarity. If using a label, wrap input and label in a div with class="form-control", which has a default display property of "flex" and flex-direction of "column".
Date input: By default, input elements inherit font styling from their parent. Be sure to use a label for clarity. Wrap input and label in a div with class="form-control", which has a default display property of "flex" and flex-direction of "column".
Parchment background: Add class="parchment" to any div to add a parchment background and turn the text inside black and bold. Also adds 0.5em of padding all around and a border-radius of 4px.
